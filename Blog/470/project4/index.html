Have you ever wanted to create a lamp that changes colors based on the light it detects? With a few simple components and a bit of coding, you can create a fascinating color mixing lamp using photoresistors!

Project Overview

The Color Mixing Lamp project is designed to change colors based on external lighting conditions. At its core, the project uses three photoresistors, each paired with a colored gel (red, green, and blue). These photoresistors change their resistance based on the amount of light they receive. The colored gels ensure that each photoresistor primarily detects light of a specific color, allowing the system to gauge the relative color levels in the ambient light.

Technical Insights:

The project requires the following setup:

  • Voltage Division: The photoresistors are paired with 10-kilohm resistors to form a voltage divider. The voltage at the junction between the photoresistor and the resistor changes based on the resistance of the photoresistor, which in turn is influenced by the light it receives.
  • Pulse Width Modulation (PWM): To achieve the fading effect of LEDs, the project employs PWM. This technique involves rapidly turning the output pin high and low, creating an illusion of varying voltage levels. The Arduino Uno, which is central to this project, has six pins dedicated to PWM.
  • LED Color Control: The project uses an RGB LED with separate red, green, and blue elements. By controlling the intensity of each color, a wide range of colors can be produced.

Reflections and Observations:

  • The LED's fading is not linear. At about half brightness, the LED appears to stop getting much brighter. This non-linearity is due to the human eye's perception of brightness.
  • The project offers insights into how sensors can be used to interact with the environment. For instance, this setup could potentially be used to indicate outdoor weather conditions while one is indoors.
  • The project also touches upon the importance of calibration. The photoresistor's output might not span the entire 0 to 1023 range, and understanding this can be crucial for more advanced applications.

Code Insights

The code initializes the serial port and sets the digital pin directions. It reads the temperature sensor's output and sends the values to the computer. The analogRead() function captures the voltage reading on the pin, which ranges between 0 and 1023. This value represents the voltage on the pin. The Serial.print() function is employed to send information from the Arduino to a connected computer.

Conclusion

The Color Mixing Lamp project isn't just about creating a visually appealing lamp, but also about understanding the underlying principles of light, color, and electronics. Whether you're a seasoned Arduino enthusiast or a curious beginner, this project offers a blend of learning and fun.
